Using [Last.FM API](https://www.last.fm/api/) and [Youtube Data API](https://developers.google.com/youtube/v3/)
to generate lists of similar artists.

Clone the repository, then install the
dependencies:

```
npm install
# or
yarn
```

To run the project:

```
yarn start
# and
react-native run-android
```
